# Taller 2 - Citas Famosas

Aplicación Ionic/Angular para gestionar citas famosas.

## Requisitos

- Node.js 18+
- npm

## Instalación

```bash
cd taller2
npm install
```

## Ejecutar

```bash
npm start
```

Abrir en: http://localhost:8100

## Estructura

- **Páginas**: Inicio, Citas, Configuración
- **Componentes**: CitaCard, FormularioCita, ListaCitas
- **Servicios**: CitasService, ConfiguracionService, DatabaseService
- **Persistencia**: IndexedDB (citas), Preferences (configuración)
